# ip-tracer
This Tool will trace someone's ip address

Steps to run this tool:
       1. git clone https://github.com/krishpranav/ip-tracer
       2. cd ip-tracer
       3. sudo chmod 777 *
       4. python3 -m pip install -r requirements.txt
       5. python3 ip-tracer.py
